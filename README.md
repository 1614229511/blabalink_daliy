# 作用
进行10个点赞, 五个帖子浏览, 之后领取每日奖励
登录的是blablalink繁体中文网页
# 用法
- Git clone项目后, pip install -r requirement安装依赖
- 配置环境变量运行, pycharm运行需要带环境变量ACCOUNT, PASSWORD, 区服只适配了日本/韓國/北美/東南亞/全球
- py main.py执行
